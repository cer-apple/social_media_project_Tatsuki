# 📱 Feed - Social Media Project

A Django-based social media platform (Instagram-inspired) built for **Advanced Python Online (5322)** at Baylor University.

## ✨ Features

- 🔐 **User Authentication** — Register, log in, and log out
- 👤 **Profile Management** — Edit your name, email, date of birth, and bio
- 📸 **Posts** — Share photos with descriptions
- 👥 **Friend System** — Send and accept friend requests
- 📰 **Dual Feed** — See your own posts or your friends' posts
- ❤️ **Engagement** — Like and comment on posts
- 🎨 **Custom Home Page** — Designed with a gradient hero section, feature cards, and smooth hover effects

## 🛠 Tech Stack

- **Backend:** Django 4.2, Python 3.13
- **Frontend:** Bootstrap 4, Crispy Forms, custom CSS
- **Database:** SQLite (development) / PostgreSQL (production on Heroku)
- **Deployment:** Heroku (Gunicorn + Whitenoise)
